# 🚀 간단 실행 가이드

## AWS EC2에서 바로 실행하기

### 1단계: EC2 접속
```bash
ssh -i your-key.pem ubuntu@your-ec2-ip
```

### 2단계: 프로젝트 가져오기
```bash
# Git 사용
git clone <your-repo-url>
cd <project-directory>

# 또는 파일 업로드 후
cd <project-directory>
```

### 3단계: 실행! (끝)
```bash
chmod +x deploy.sh
./deploy.sh
```

**끝입니다!** 🎉

모든 것이 자동으로 처리됩니다:
- 환경 변수 자동 생성
- SECRET_KEY 자동 생성
- EC2 IP 자동 감지
- Docker 자동 설치
- 서비스 자동 시작

## 접속하기

배포가 완료되면 자동으로 표시되는 주소로 접속:

- **프론트엔드**: `http://your-ec2-ip`
- **백엔드 API**: `http://your-ec2-ip:8000`
- **API 문서**: `http://your-ec2-ip:8000/docs`

## 문제가 생기면?

```bash
# 로그 확인
docker-compose -f docker-compose.prod.yml logs -f

# 재시작
docker-compose -f docker-compose.prod.yml restart

# 다시 배포
./deploy.sh
```

## 더 간단하게?

Docker Compose만 실행해도 됩니다:

```bash
docker-compose -f docker-compose.prod.yml up -d
```

환경 변수는 자동으로 기본값이 사용됩니다!
