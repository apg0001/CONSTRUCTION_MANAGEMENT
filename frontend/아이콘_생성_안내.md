# 🎨 PWA 아이콘 생성 안내

## 현재 상태

PWA 설정은 완료되었지만, 실제 아이콘 이미지 파일이 필요합니다.

## 필요한 파일

다음 파일들을 `frontend/public/` 디렉토리에 추가해야 합니다:

- `icon-192.png` (192x192 픽셀)
- `icon-512.png` (512x512 픽셀)

## 아이콘 생성 방법

### 방법 1: 온라인 도구 사용 (가장 쉬움)

1. **PWA Asset Generator**
   - https://github.com/elegantapp/pwa-asset-generator
   - 또는 https://www.pwabuilder.com/imageGenerator
   - 이미지 업로드 → 자동 생성

2. **RealFaviconGenerator**
   - https://realfavicongenerator.net/
   - 다양한 플랫폼 아이콘 자동 생성

### 방법 2: 직접 생성

1. **이미지 준비**
   - 512x512 픽셀 이상의 정사각형 이미지
   - 투명 배경 또는 단색 배경

2. **리사이즈**
   - 192x192 픽셀로 리사이즈 → `icon-192.png`
   - 512x512 픽셀로 리사이즈 → `icon-512.png`

3. **저장 위치**
   ```
   frontend/public/icon-192.png
   frontend/public/icon-512.png
   ```

### 방법 3: 간단한 아이콘 (임시)

현재 `icon.svg` 파일이 있습니다. 이를 PNG로 변환:

```bash
# ImageMagick 사용 (설치 필요)
convert -background none -size 192x192 icon.svg icon-192.png
convert -background none -size 512x512 icon.svg icon-512.png
```

## 빠른 테스트용 아이콘

임시로 간단한 아이콘을 만들려면:

1. 온라인 도구 사용:
   - https://www.favicon-generator.org/
   - 텍스트 "건설" 입력
   - 생성된 파일 다운로드

2. 또는 기본 아이콘 사용:
   - 브라우저 기본 아이콘이 표시됨
   - 나중에 커스텀 아이콘으로 교체 가능

## 확인 방법

아이콘 추가 후:

1. 빌드: `pnpm build`
2. 브라우저에서 접속
3. 개발자 도구 > Application > Manifest
4. 아이콘이 표시되는지 확인

## 중요 사항

- ✅ 아이콘 파일은 반드시 `public/` 디렉토리에 있어야 함
- ✅ 파일 이름은 정확히 `icon-192.png`, `icon-512.png`여야 함
- ✅ PNG 형식 권장
- ✅ 투명 배경 또는 단색 배경 권장
